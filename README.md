# 🔐 Password Fortress

A modern and secure password strength analyzer built using HTML, CSS, and JavaScript.

---

## ✨ Features

* 🔒 Real-time password strength analysis
* 🎲 Strong password generator
* 📋 Copy password feature
* 🌙 Dark / Light mode toggle
* 📊 Advanced entropy-based scoring system
* ✅ Security checkpoints validation
* ⚡ Smooth animations and modern UI
* 📱 Fully responsive design
* 🔐 100% local processing (No password storage)

---

## 🚀 Technologies Used

* HTML5
* CSS3
* JavaScript
